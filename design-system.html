design_services 디자인 시스템 개요

Weekly Quest 디자인 시스템은 일관된 사용자 경험과 시각적 정체성을 제공하기 위한 가이드라인입니다. 이 문서는 개발자와 디자이너가 Weekly Quest 프로젝트의 UI 요소를 구현하고 유지하는 데 참조할 수 있는 표준을 정의합니다.

디자인 원칙
palette

일관성

모든 UI 요소는 일관된 디자인 언어를 따릅니다.

accessibility

접근성

모든 사용자가 쉽게 이용할 수 있도록 설계되었습니다.

성능

빠르고 효율적인 사용자 경험을 제공합니다.

devices

반응형

모든 디바이스에서 최적의 경험을 제공합니다.

palette 색상 시스템

Weekly Quest의 색상 시스템은 일관된 시각적 정체성을 유지하고 접근성을 보장하기 위해 설계되었습니다. 모든 색상은 라이트 모드와 다크 모드에서 최적의 가독성을 제공합니다.

기본 색상

애플리케이션의 주요 색상 팔레트입니다. 이 색상들은 일관된 브랜드 아이덴티티를 유지하는 데 사용됩니다.

text_fields 타이포그래피

타이포그래피는 사용자 인터페이스의 핵심 요소입니다. Weekly Quest는 명확한 텍스트 계층 구조를 통해 가독성과 사용자 경험을 강화합니다.

제목 스타일

제목은 콘텐츠의 계층 구조를 정의하고 사용자가 콘텐츠를 쉽게 탐색할 수 있도록 도와줍니다.

제목 1 (Heading 1)

큰 섹션의 제목이나 페이지 제목에 사용합니다.

h1 { font-size: var(--text-2xl); font-weight: 700; line-height: 1.2; }

제목 2 (Heading 2)

섹션 내 하위 제목이나 카드 제목에 사용합니다.

h2 { font-size: var(--text-xl); font-weight: 600; line-height: 1.3; }

제목 3 (Heading 3)

h3 { font-size: var(--text-lg); font-weight: 600; }

본문 텍스트 (Body Text)

.body-text { font-size: var(--text-base); line-height: 1.6; }

작은 텍스트 (Small Text)

.text-sm { font-size: var(--text-sm); }

부가 설명 텍스트 (Muted Text)

.text-muted { color: var(--text-muted); }

카드

카드는 콘텐츠를 그룹화하고 구조화하는 데 사용됩니다. 다양한 스타일을 제공하여 디자인 언어를 풍부하게 합니다.

카드 유형

기본 카드

기본적인 카드 스타일입니다. 단순하면서도 효과적인 디자인을 제공합니다.

뉴모피즘 카드

입체적인 그림자 효과가 적용된 카드입니다. 모던하고 세련된 느낌을 줄 수 있습니다.

글래스모피즘 카드

반투명한 유리 효과가 적용된 카드입니다. 현대적이고 세련된 느낌을 줄 수 있습니다.

  <p>카드 내용...</p>
  <button class="btn-text mt-4">자세히 보기</button>
</div>

space_bar 간격 시스템
